# Login Screen Wireframe

## Project Overview

This project contains low-fidelity mobile login screen wireframes created using Figma.

The purpose of this design is to demonstrate the UI/UX process for a simple authentication flow.

---

## Features

- Login Screen
- Sign Up Screen
- Forgot Password
- Responsive Mobile Layout
- Simple User Flow
